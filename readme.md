# Projeto #2: Site Profissional (HTML/CSS) - Repositório acadêmico voltado ao aprendizado.
Este é o meu segundo projeto prático focado nos fundamentos do desenvolvimento web front-end. O objetivo foi aplicar conceitos essenciais de estruturação e estilização, criando um site online, limpo e bem organizado.

# 🚀 Tecnologias Utilizadas
HTML5: Estruturação do conteúdo.
CSS3: Estilização, layout e responsividade básica.
VS Code: Editor de código utilizado para o desenvolvimento.
